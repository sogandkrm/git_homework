Git Commands Log
=================

This file contains a list of Git commands executed during the Git Homework project.

--- Repository Setup ---
git init
git status
git config --global user.name
git config --global user.email

--- Basic Workflow ---
git add .
git add <file>
git commit -m "message"
git log
git log --oneline

--- Branching ---
git branch
git branch -m main
git checkout <branch>
git checkout -b <branch>
git switch <branch>
git switch -c <branch>

--- Merging ---
git merge <branch>
git merge --abort

--- Stash ---
git stash
git stash list
git stash apply
git stash pop

--- Diff and Inspection ---
git diff
git diff --staged

--- Reset and Revert ---
git reset --soft HEAD~1
git reset --mixed HEAD~1
git reset --hard HEAD~1
git revert HEAD
git revert <commit-hash>

--- Remote and GitHub ---
git remote -v
git remote add origin <repo-url>
git push
git push -u origin main
git pull
git fetch

--- Branch Cleanup ---
git push origin --delete master
